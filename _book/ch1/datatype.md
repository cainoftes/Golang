# 数据类型

## 整数类型

| Length   | Signed | Unsigned |
| -------- | ------ | -------- |
| 8-bit    | int8   | uint8    |
| 16-bit   | int16  | uint16   |
| 32-bit   | int32  | uint32   |
| 64-bit   | int64  | uint64   |
| 基于架构 | int    | uint     |

## 浮点型

| Length | 32-bit    | 64-bit     |
| ------ | --------- | ---------- |
| 浮点数 | float32   | float64    |
| 复数   | complex64 | complex128 |

## 布尔类型

bool

## 字符类型

+ 代表 ASCII 的 byte（== uint8）
+ 代表 UTF-8 的 rune（== int32）
