# 常量定义

## 常量定义

> 常量定义非常类似于变量，只是 var 变成了 const，同时声明的时候必须初始化

```go
const pi = 3.14
const (
	n1 = 100
    n2 // 和上一行相同
    n3 // 和上一行相同
)
```

## itoa

+ 可以视为一个常量值，只能在常量的表达式中使用
+ 在 const 出现的时候 itoa 被置为0
+ 每新增一行都会增加1

举几个栗子

+ 普通使用

  ```go
  const (
  	n1 = iota // 0
      n2 // 1
      n3 // 2
      n4, n5 = iota, iota + 2 // 3, 5
      n6 // 
  )
  ```

+ 使用 _ 跳过值

  ```go
  const (
  	n1 = iota
      n2 // 1
      _
      n4 // 4
  )
  ```

+ 中间插队

  ```go
  const(
  	n1 = iota // 0
      n2 = 100
      n3 = iota // 2
  )
  ```
